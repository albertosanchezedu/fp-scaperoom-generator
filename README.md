## ¿Qué consigues?

Una actividad lista para tu aula virtual donde los alumnos juegan en equipo resolviendo puzzles, descifran códigos y acaban compitiendo individualmente en un cuestionario que se autocorrige y envía la nota al calificador.

## ¿Cómo funciona?

1. Abre el generador y pega el contenido de tu tema
2. Pulsa **Generar prompt** — se crea automáticamente un prompt optimizado para Claude
3. Pégalo en [claude.ai](https://claude.ai), revisa el plan de salas y pide los archivos
4. Sube `escape_room.html` a Moodle como **Archivo** e importa `cuestionario.txt` como cuestionario **AIKEN**

## Características

- Vidas, temporizador, sala de castigo y códigos cifrados con César
- Bases de datos manipuladas en cada sala para obligar a usar los apuntes
- Los alumnos juegan en equipo — cada uno completa el cuestionario individualmente
- Guía paso a paso de instalación en Moodle incluida
- Compatible con **Moodle**, **EducaMadrid** y **EVEX Extremadura**
